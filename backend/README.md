# Backend: Smart wardrobe

## Описание

Приложение FastAPI для управления гардеробом.

## Быстрый запуск (Backend)

1. Установите зависимости:
   ```bash
   cd backend
   uv sync
   ```
2. Запустите сервер:
   ```bash
   uv run uvicorn main:app --reload
   ```
3. Откройте http://localhost:8000/docs
